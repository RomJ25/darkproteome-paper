# manuscript/ — what is canonical, what is not

- **`manuscript.md`** — the **CANONICAL paper**. Everything else here is derived from it.
- **`figures/`** — Figs 1–4 (PDF + PNG), generated by `make_figures.py`.

**Build artifacts — do NOT hand-edit (regenerate instead):**
- `manuscript.html` and `manuscript.pdf` (the latter untracked) are generated from `manuscript.md`.
  After editing `manuscript.md`, regenerate: `python3 manuscript/md_to_html.py` (then Print → PDF).
- `tex/` — the LaTeX (tectonic) typeset build of the paper: `manuscript.tex` (wrapper, also
  defines the math macros) + `_body.tex` (the `manuscript.pdf` / `_preview` / aux outputs are
  gitignored). Rebuild with `tectonic manuscript/tex/manuscript.tex --outdir manuscript/tex`,
  then copy the resulting PDF to `manuscript/manuscript.pdf`. Uses the XeTeX-native `libertinus`
  package (Libertinus Serif + unicode-math).
