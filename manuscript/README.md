# manuscript/ — what is canonical, what is not

## ✅ The paper

- **`manuscript_v2.md`** — the canonical paper: *"Extensive canonical-sequence overlap and unresolved
  source attribution in a public non-canonical HLA-peptide atlas and its search library."*
- **`supplement_v2.md`** — S1 (pseudogene→parent homology) · S2 (the class-FDR derivation, reproduced
  from Woo et al. 2014 — not ours).
- **`figures_v2/`** — Figures 1–3. Generated by `make_figures_v2.py`, which reads the JSON artifacts in
  `data/` rather than typed-in numbers, so a figure cannot drift from the text.
- **`verify_manuscript.py`** — the build gate. It re-derives every headline number from the committed
  artifacts and **fails on drift**, refuses 28 retracted phrasings, and enforces the ORF-class partition
  arithmetically. Checks that need the large inputs (not redistributed here) are reported as *skipped*
  rather than crashing.

```bash
python3 manuscript/verify_manuscript.py
```

## 🔴 The withdrawn paper — do not cite, quote, or build

- **`manuscript.md`** — **WITHDRAWN.** Its headline (*"0 of 306,844 claims pass all four evidence
  axes"*) was **vacuous — true by construction**: the scorer could only record a pass via statistics
  that no source publishes, so no claim *could* pass. A second defect scored *unassayed* claims as
  immunogenicity **failures**. See **`REVISION_NOTICE.md`**. It is retained, unaltered and clearly
  marked, so the correction is inspectable — not so it can be cited.
- **There is deliberately no `manuscript.pdf`.** A PDF cannot be updated once it has been downloaded,
  and a PDF is what gets cited. Publishing an unmarked PDF of a withdrawn paper is exactly how a
  retracted result outlives its retraction. The sources remain, so anyone who wants it can build it.
- `manuscript.html`, `tex/`, and the per-section `*.md` drafts are artifacts and scaffolding of the
  withdrawn paper.

`manuscript_v2.md` has no LaTeX build, by choice: hand-maintaining a parallel `_body.tex` is how a
rendered artifact silently drifts from its source.
